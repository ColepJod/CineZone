# CineZone

CineZone is a movie discovery web app built using HTML, CSS, and JavaScript. It allows users to search movies, view their ratings, and explore details like release date, overview, and posters. The project focuses on creating an attractive and user-friendly interface for movie lovers.

***

## Features

- Search movies by title  
- View movie posters, ratings, and release dates  
- Responsive design for all screen sizes  
- Clean and dynamic interface built with HTML, CSS, and JavaScript  
- Integration with a movie API (like TMDB) for real-time data  
- Option to explore trending and popular movies  

***

## Tech Stack

- HTML for structure  
- CSS for styling and layout  
- JavaScript for functionality and API handling  

***

## Setup and Installation

1. Clone the repository  
   ```bash
   git clone https://github.com/yourusername/CineZone.git
   ```
2. Navigate to the project directory  
   ```bash
   cd CineZone
   ```
3. Open the `index.html` file in your browser  

***

## Usage

- Enter any movie name in the search bar  
- Click search to view results with titles, posters, and ratings  
- Browse trending and popular movies on the home page  
- Click a movie card to view more details
